Quora.com 

התוכנה מוצאת שאלות, מייצאת את התשובות לשאלה ואת פרופילי המגיבים מהאתר quora לקובץ אקסל.
להפעלה יש למקם את הקובץ chromedriver.exe באותה תיקייה עם קובץ ה- jar.

יש להכניס לשדה keywords את מילות המפתח לשורת החיפוש באתר.

שדה topics משמש לתגיות והנושאים בהם מילות המפתח עוסקים, ע"מ למקד את החיפוש. יש להפריד בין נושא לנושא ע"י פסיק ','.
הוספת topic תמקד את החיפוש אך פחות תוצאות יופיעו בעמוד התוצאות.

number of question -מספר השאלות המבוקש (שייוצא לאקסל). ברירת המחדל (להשאיר ריק) יביא 10,000 שאלות, או יפעל עד סוף התוצאות.
כאשר משתמשים בחיפוש ללא topics מומלץ להגביל את החיפוש שכן יכול להיות כמות גדולה של תוצאות.

number of answers - מספר התשובות המבוקש לכל שאלה שנבחרה. ברירת המחדל היא 10,000 או יביא את כל התשובות לשאלה. גם שדה זה מומלץ להגביל שכן כאשר מדובר בשאלה פופולרית, מספר התשובות יכול להיות גדול מאוד.

minimum (number of) follows עובד per question. ציון מינימום עוקבים עבור שאלה שתיוצא לאקסל. ז"א כל שאלה בתוצאות עם מספר עוקבים קטן מהמצוין, התוכנה תדלג לשאלה הבאה.
minimum upvotes ו -minimum views עובד per answer. ציון מינימום צפיות ומינימום הצבעות מעלה עבור כל תשובה לשאלה. כל תשובה לא פופולרית מספיק (מתחת למינימום המצוין) התוכנה לא תייצא לקובץ האקסל.
